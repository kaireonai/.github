KaireonAI determines the best action for every customer in real time. Define business rules visually, score candidates with ML models, enforce contact policies, run experiments, and measure uplift.

### What it does

- **Visual Decision Pipelines** — Drag-and-drop flows with 16 node types: qualify, score, rank, enrich, filter, and more
- **8 ML Scoring Engines** — Scorecard, Bayesian, Gradient Boosted, Thompson Bandit, Neural CF, and more. All train on real interaction data.
- **Real-Time Recommendations** — Sub-200ms API response with PRIE multi-objective scoring
- **Contact Policies** — Frequency caps, cooldowns, mutual exclusion. Prevents customer fatigue automatically.
- **Experiments** — Champion/challenger A/B testing with statistical uplift analysis
- **24 Data Connectors** — S3, Kafka, Snowflake, BigQuery, PostgreSQL, and more
- **AI Assistant** — 110 MCP tools for natural-language platform management

### Get started in 2 minutes

```bash
git clone https://github.com/kaireonai/platform.git
cd platform
docker compose up
```

Visit `http://localhost:3000`, register, and your workspace auto-loads with the Starbucks demo dataset — 10 offers, 6 channels, 3 ML models, ready to explore.
